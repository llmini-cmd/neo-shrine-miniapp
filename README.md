# NEO SHRINE

Farcaster上で遊べる、BASEをお賽銭にした Web3 おみくじゲーム。  
ゲーム体験・アニメーション・音の反応を通して、おみくじを獲得し、  
結果をギャラリーとして蓄積・共有できるミニアプリです。

---

## 1. プロジェクト概要

NEO Shrine は、Farcasterを起点に遊べるミニゲーム型のおみくじ体験です。  
ユーザーはゲームをプレイし、BASEチェーンのETHを「お賽銭」に見立てて投げることで、
おみくじ結果を獲得します。

単なる結果表示ではなく、

- タップへの反応
- アニメーション
- 音・タイミング
- 視覚的フィードバック

を重ねることで、「遊んだ結果としておみくじが残る」体験を重視しています。

将来的には、おみくじ結果やギャラリーをNFTとして展開することも想定していますが、
現時点では **ゲーム体験を最優先** とした設計になっています。

---

## 2. 体験コンセプト

- Farcaster上で気軽に遊べる
- Web3要素を“操作”ではなく“体験”として感じられる
- 結果だけでなく、遊ぶ過程そのものが楽しい
- 何度も引きたくなる、集めたくなる

NEO Shrine は  
**「ゲーム → おみくじ → ギャラリー → シェア」**  
という一連の流れを、1つの体験として設計しています。

---

## 3. ゲームフロー

1. **ゲーム開始**
   - タップ操作を中心としたミニゲーム
   - 音ゲー要素を意識したタイミング設計

2. **プレイ中のフィードバック**
   - タップに応じて鈴などのモチーフが反応
   - PERFECT / GREAT などの評価を表示
   - すべての要素がアニメーションで反応

3. **お賽銭アクション**
   - BASEのETHを「投げる」動作として表現
   - 単なる支払いではなく、行為として体験させる設計

4. **おみくじ結果表示**
   - 結果はカード形式で表示
   - 視覚的に「獲得した」ことが分かる演出

5. **ギャラリー保存**
   - 獲得したおみくじはギャラリーに蓄積
   - いつでも見返すことができる

6. **シェア**
   - Farcasterへ結果をシェア
   - プレイ体験の延長として自然に共有可能

---

## 4. UI / UX 設計方針

### 4.1 アニメーション重視の設計

- すべてのボックス・カード・ボタン・装飾要素に動きを持たせる
- 「触ると反応する」ことを前提にUIを設計
- 派手すぎず、テンポと気持ちよさを最優先

### 4.2 タップ体験の作り込み

- タップのタイミングに応じた評価（PERFECT / GREAT）
- 視覚・音・動きが同時に返ってくる設計
- 短時間でも没入できる感覚を重視

### 4.3 お賽銭という解釈

- ETHを単なる通貨として扱わない
- 「投げる」「納める」という動作として表現
- Web3要素を体験の中に自然に溶け込ませる
