# NEO Shrine

> Farcasterで遊べる、BASEをお賽銭にした“動きで楽しむ”Web3おみくじゲーム。

## 概要

NEO Shrine は、Farcaster上で気軽に遊べるミニゲーム型のおみくじ体験です。  
ユーザーはゲームをプレイし、BASEのETHをお賽銭に見立てて投げることでおみくじを獲得できます。

画面内のすべてのボックスやコンポーネントにアニメーションを取り入れ、  
「タップすると反応する」「動きが返ってくる」体験を重視しています。  
ゲーム性・演出・結果表示を一連の流れとして設計し、  
おみくじ結果はギャラリーとして保存・閲覧できます。

お正月シーズンには「w」をモチーフにした Web3 大喜利おみくじも用意しており、  
ゲームから大喜利までをまとめて楽しめる構成になっています。  
将来的には、これらのおみくじを NFT として展開することも想定しています。

## 推しポイント

1. **すべてのUIコンポーネントに動きを持たせた設計**
   - ボックス、カード、ボタン、装飾要素など、画面内のほぼすべてにアニメーションを実装。
   - 置かれているだけのUIではなく、「触ると必ず反応する」ことを前提にしています。
   - 派手すぎず、テンポと気持ちよさを重視した調整を行っています。

2. **BASEのETHを“お賽銭”として投げる動作へのこだわり**
   - ETHの支払いを単なる操作にせず、「お賽銭を投げる」という動きとして表現。
   - 投げるモーション、タイミング、視覚的なフィードバックを細かく調整しています。
   - 金額の大小ではなく、参加している感覚を重視しています。

3. **音ゲーを意識したミニゲーム体験**
   - タップに応じて鈴などのモチーフが反応。
   - プレイ結果は **PERFECT / GREAT** などの評価でフィードバックされます。
   - 短時間でも没入できる、音とタイミングを重視した設計です。

4. **おみくじを保管・閲覧できるギャラリー**
   - 引いたおみくじはギャラリーとして蓄積。
   - 初期状態は空でも問題なく、徐々に埋まっていく体験を大切にしています。
   - コレクションを眺める楽しさも含めて設計しています。

5. **Web3大喜利おみくじという遊び心**
   - 「w」をモチーフにした Web3 大喜利形式のおみくじを用意。
   - Web3カルチャーを知っている人ほど楽しめる、軽い笑いの要素を取り入れています。
   - ゲーム → おみくじ → 大喜利 → シェアまでが一連の流れとして完結します。

6. **将来のNFT展開を見据えた構成**
   - 現在はゲーム体験を最優先。
   - おみくじ結果やギャラリー構造は、将来的にNFTとして展開しやすい形で設計しています。
   - 今は遊びやすさを重視し、段階的な拡張を想定しています。

## 使用技術（こだわりポイント）

- **フロントエンド**:
  - WebベースのミニゲームUI
  - アニメーションを多用したインタラクション設計
- **バックエンド**:
  - 軽量API構成（体験を阻害しないことを最優先）
- **データベース**:
  - おみくじ結果・ギャラリー管理用（最小限）
- **インフラ**:
  - Webホスティング
- **その他**:
  - Farcaster上での利用を前提にしたUX設計
  - BASEチェーンのETHを「お賽銭」として扱うコンセプト設計
